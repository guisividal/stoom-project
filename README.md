#

##TECNOLOGIAS
* JAVA 11
* Spring Boot 2.4.x
* Spring Data JPA
* Postgres SQL 

## RODANDO sem Builder

* `mvnw spring-boot:run`

## COMO BUILDA & Executar

* `mvnw clean package`
* `cd target`
* `java -jar stoom-project-0.0.1-SNAPSHOT.jar`

## Executar somente testes

- necessita postgres e internet

* `mvnw test`
